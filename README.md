# Usage

    git clone git@github.com:sputnick-dev/sedtris.git
    cd sedtris
    ./sedtris.sh

or 

    wget https://github.com/sputnick-dev/sedtris/archive/refs/heads/main.zip
    unzip main.zip
    cd sedtris-main/
    chmod +x sedtris.sh
    ./sedtris.sh
